Representations of Bn and Dn

class bn.RepBn(highest_weight_numerator, denominator)
class dn.RepDn(highest_weight_numerator, denominator)

Creates the representation object with highest weight highest_weight/denominator . The parity of the entries in the numerator should be the same.

The highest weight of a representation object.

bn.RepBn.get_root_vector(root, normalization = 'rational')
dn.RepDn.get_root_vector(root, normalization = 'rational')

Returns the root vector \(E_{\alpha}\) in a Chevalley basis cooresponding to the root \(\alpha\). The root \(\alpha\) needs to be represented as a list of integer coefficients in the simple root basis. The option normalization = 'rational' will make the entries of root vectors rational, while the option normalization = 'unitary' will make the representation restricted on the maximal compact subgroup \(SO(n)\) (realized as orthogonal matrices in \(GL(n,\mathbb{C})\)) unitary. The computed root vectors are stored in the dictionary an.RepAn.root_vectors_rational and an.RepAn.root_vectors_unitary, respectively.

The transformation matrix from the rational form to the unitary representation is diagonal, and the diagonal elements can be computed with the method

bn.RepBn.unitarization()
dn.RepDn.unitarization()

The result of the computation is stored in the attribute bn.RepBn.unitarization_matrix and dn.RepDn.unitarization_matrix, respectively.
