Gelfand-Tsetlin Basis of Type An

A Gelfand-Tsetlin pattern for type \(A_{n-1}\) is a pyramid of (nonnegative) integers \[\begin{array}{ccccccc} \lambda_{n,1} & & \lambda_{n,2} & & \ldots & & \lambda_{n,n}\\ & \lambda_{n-1,1} & & \ldots & & \lambda_{n-1,n-1}\\ & & \ldots & & \ldots\\ & & \lambda_{2,1} & & \lambda_{2,2}\\ & & & \lambda_{1.1} \end{array}\] that satisfies the interlacing relation \[\lambda_{k,i} \geq \lambda_{k-1,i} \geq \lambda_{k,i+1}.\] The Gelfand-Tsetlin pattern records the branching of an irreducible representation with highest weight (as a partition)\[\lambda =(\lambda_{n,1}, \lambda_{n,2} ,\ldots,\lambda_{n,n} )\] along the chain of subgroups \[GL(n)\supset GL(n-1)\supset\ldots\supset GL(1).\]

class an.GTPattern(GT_pattern_as_list)

A Gelfand-Tsetlin pattern is usually stored as a list of rows. We can also use the class GTPattern to create a Gelfand-Tsetlin pattern object for convenience. The pattern is stored in the attribute an.GTPattern.pattern as a list. For example:

> b = GTPattern([[3, 2, 1], [2, 1], [1]])
> print(b.pattern)

[[3, 2, 1], [2, 1], [1]]

an.GTPattern.get_lambda(k, i)

Returns the entry \(\lambda_{k,i}\) for each pattern object.

an.GTPattern.get_l(k, i)

Returns the value of \(l_{k,i} = \lambda_{k,i} - i +1\) for each pattern object.

an.GTPattern.raise_pattern(k, i)
an.GTPattern.lower_pattern(k, i)

Raise/lower the entry \(\lambda_{k,i}\) pattern object by 1, respectively. Caution: These two methods do not have return values and will modify the original instance of GTPattern.

an.GTPattern.raised(k, i)
an.GTPattern.lowered(k, i)

Return new GTPattern objects with the entry \(\lambda_{k,i}\) raised/lowered by 1, respectively.

Caution: The methods provided in this class do not check the interlacing condition. The user will have to provide their own method for that purpose.
