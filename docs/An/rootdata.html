Documentation for LieGT

Type An

class an.RootDatAn(n)

Creates the root data object for \(A_n\).

an.RootDatAn.rank

The rank of the root system.

an.RootDatAn.roots

The list of roots of \(A_n\) in the usual basis \(e_i - e_j\) for \(i\neq j\). Roots are stored as sympy.Matrix objects.

an.RootDatAn.simple_roots

The list of simple roots of \(A_n\) in the usual basis \(e_i - e_{i+1}\) for \(1\leq i \leq n\). 

an.RootDatAn.coroots

Returns the list of coroots \(\check\alpha = \frac{2\alpha}{\langle\alpha,\alpha\rangle}\)of \(A_n\) in the usual basis.

an.RootDatAn.simple_coroots

Returns the list of simple coroots in the usual basis.

an.RootDatAn.cartan_matrix

Returns the Cartan matrix.

an.RootDatAn.fundamental_weights

Returns the list of fundamental weights. Fundamental weights are stored as sympy.Matrix objects.

an.RootDatAn.root_strings

Returns the list of roots in the fundamental weight basis. They are stored as lists with integer entries.

an.RootDatAn.positive_root_strings

Returns the list of positive roots in the fundamental weight basis. They are stored as lists with integer entries.

an.RootDatAn.highest_root_string

Returns highest root in the fundamental weight basis.

an.RootDatAn.highest_root

Returns the highest root in the usual basis.

an.RootDatAn.root_order

Returns the dictionary of a lexiographic order of roots. Roots are stored as tuples.

an.RootDatAn.positive_root_order

Returns the dictionary of a lexiographic order of positive roots. Roots are stored as tuples.

an.RootDatAn.get_p(root_string_alpha, root_string_beta)

Returns the greatest positive integer \(p_{\alpha,\beta}\) such that \(\beta-p\alpha\) is a root.

an.RootDatAn.get_epsilon(alpha, beta)

Returns a choice of \(\epsilon_{\alpha,\beta}=\pm 1\) in the commutation relation \([E_\alpha, E_\beta]=\epsilon_{\alpha,\beta}(p_{\alpha,\beta}+1)E_{\alpha+\beta}\) of the Chevalley basis.
